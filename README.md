# utils
---
folder_migrate:
	A tool to migrate local directories to remote servers. Uses ssh and rsync tool.

tree_creator:
	Creates directories and files based on text tree structures.
	```
	flashcard-app/
	├── backend/
	│   ├── main.py
	│   ├── models.py
	│   └── requirements.txt
	└── frontend/
	    ├── src/
 	   │   ├── App.js
 	   │   └── Flashcard.js
	    └── package.json

	```
